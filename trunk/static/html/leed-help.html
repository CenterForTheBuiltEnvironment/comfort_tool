With this feature of the tool you can feed the results of thermal calculations to a separate compliance document, for different conditioning modes and seasons.

The steps to use this dialog are the following:

  1. Type the name of the space you are evaluating.

  2. Select cooling or heating mode.

  3. Make the appropriate changes in the input parameters for this space and system.

  4. Set these parameters as the appropriate season with the set season buttons.

    (note: you can make changes and re-click the season buttons as many times as you want, overwriting the data for the current space and system)

  5. Repeat this process for each space type, system, and season until you are finished.

  6. Click on "view document" to open the compliance page.

    (note: be sure to allow pop-ups in your browser, or else the page will not be visible)

  7. If you wish to start over, simply refresh your browser.

  8. Once you are done and the page is open, you can print it as pdf to have a copy on your desktop.
